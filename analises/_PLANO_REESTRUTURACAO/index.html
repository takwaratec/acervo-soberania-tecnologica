    📇 ETAPA 2: PADRONIZAR CITAÇÕES ABNT

    As 106 fichas com citação genérica precisam ser corrigidas. São 3 tipos:

    Tipo A — "Ficha XXX. 2026." (fichas MQTF legadas) - Extrair título real do corpo da ficha (seção "Título original:" ou "Título:") - Se não houver, usar o nome do arquivo como título

    Tipo B — "A CAVICHIOLI. 2026." (conversões automáticas sem metadados) - Verificar se o PDF original tem dados - Se não, marcar como ⏳ Pendente de dados

    Tipo C — Fichas novas da esteira (hoje) - Têm texto extraído mas sem autor/DOI - Manter como estão — a revisão manual vai preencher

    Script de correção: 

    # Para cada ficha com how_to_cite genérico:
# 1. Buscar título real no corpo (seção 1 ou 3)
# 2. Buscar ano no corpo
# 3. Gerar citação: "{titulo_real}. {ano}."

    Etapa O que Tempo estimado
    1 Mover 658 fichas para subgrupos ~15 min (script)

    ⚠️ RISCOS

    • Links quebrados: fichas que referenciam outras pelo caminho relativo vão quebrar com a mudança de diretório
    • Git: mover arquivos = git mv, o histórico é preservado
    • ChromaDB: precisa reindexar após mover — o embedding é baseado no conteúdo, não no caminho, então não perde dados

    Mitigação: o script de migração deve atualizar os links internos (Conexões, seção 6) que usam ../ nos paths relativos.
