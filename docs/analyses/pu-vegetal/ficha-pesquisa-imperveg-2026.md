## 3. Resumo

Relatório compilado que reúne e sintetiza pesquisas acadêmicas sobre aplicações do poliuretano vegetal de mamona (Imperveg) como adesivo e matriz em compósitos. Abrange os estudos de:

1. **Melo (2023)** — Compósito de eucalipto com Imperveg AGT 1315 para reparo de fendas (já fichado como resenha-melo-2023)
2. **Cazella (~2022)** — Painéis Pinus + PET com adesivo AGT 1315 (fichado nesta safra)
3. **Mariana Lopes (~2020)** — Compósitos de bambu para OSB (fichado nesta safra)
4. **Oliveira (~2019)** — Painéis eucalipto + couro com PU de mamona (fichado nesta safra)

## 4. Análise Crítica

### Pontos Fortes
- Visão panorâmica do estado-da-arte das aplicações do Imperveg como adesivo
