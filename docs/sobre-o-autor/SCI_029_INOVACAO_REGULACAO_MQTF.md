## 3. RESUMO / SÍNTESE

![Selo de Rigor Científico](https://img.shields.io/badge/RIGOR-PVIT-green?style=for-the-badge)
![Status MQTF](https://img.shields.io/badge/STATUS-CONSOLIDADO-blue?style=for-the-badge)


O projeto MQTF utiliza a tecnologia de poliuretano vegetal que obteve as mais rigorosas certificações internacionais na União Europeia (via **Tecnoveg Portugal**). Esta conformidade garante que a inovação biotecnológica brasileira atende aos padrões de engenharia civil pesada, superando resinas fósseis tradicionais.


Abaixo, os resultados dos testes de conformidade realizados em laboratórios acreditados:

| Norma / Teste | Resultado Obtido | Significado para o Projeto |
| :--- | :--- | :--- |
| **EN 1542 (Pull-off)** | Aderência > 2.0 MPa | Superior à resistência do concreto estrutural. Garante longevidade das Biofábricas. |
| **EN 1062-6 (Permeabilidade CO2)** | Sd > 50 m | Proteção total contra carbonação. Essencial para a durabilidade de ferragens e reforços. |
| **EN ISO 7783 (Vapor de Água)** | Classe I (Permeável) | Permite que a estrutura "respire", evitando bolhas e degradação interna. |
| **EN ISO 2812-1 (Resistência Química)** | Alta Resistência | Suporta ambientes agressivos em ETEs (Estações de Tratamento de Esgoto). |
