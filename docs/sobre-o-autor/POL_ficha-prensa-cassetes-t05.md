A Prensa de Cassetes Intercambiáveis T05 é um equipamento hidráulico-mecânico multinível projetado para a conformação de elementos construtivos a partir de biocompósitos de PU vegetal e biomassa. O sistema opera com pressão nominal de **10 MPa (100 bar)**, garantindo a densidade crítica de **800 kg/m³** necessária para a estabilidade estrutural dos componentes prensados.

O diferencial tecnológico reside no **Sistema de Pistão Axial com Cassetes Intercambiáveis** — moldes rápidos que se encaixam no chassi mestre por meio de travas de engate rápido, permitindo a produção de diferentes geometrias (tijolos maciços, tijolos vazados, cobogós, telhas, blocos de vedação) no mesmo equipamento, sem necessidade de ferramentas ou ajustes complexos. O sistema admite resistências elétricas acopláveis para termoprensagem (ativação por calor), ampliando as possibilidades de formulação do biocompósito. A motorização é hidráulica monofásica de 3 CV, compatível com rede elétrica rural (110/220V).
